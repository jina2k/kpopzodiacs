This is a discontinued project that utilizes web scraping from the website kprofiles.com and redirects traffic to the same site depending on what the user's birthday is.
To look at the disclaimers regarding copyright issues, click here
The extent of web scraping in this project goes as far as copying the birthdays listed and the stage name of each person, and parsing it for ease of use.
The rest of the project was intended to store image links found from Google's search engine such as commons.wikimedia.org into an SQL database, but given the size of the data (1400 people) and the limits on Google's Custom Search API, it seemed rather inefficient to retrieve non-existent images under the Creative Commons license. (though for many well-known artists, this was not a concern)
As for the implementation, simple functions were used to determine what the user's birthday is close to.
The languages used in this project include HTML, CSS, Vanilla Javascript, PHP, and MySQL.
